# BytePlus Setup Guide

This guide is intentionally conservative. It helps you prepare, dry-run, and verify one paid task only after all safety gates pass.

1. Confirm your BytePlus account has ModelArk access.
2. Confirm the AP-Southeast region in Console.
3. Confirm `dreamina-seedance-2-0-260128` is activated.
4. Confirm billing, rate limits, and budget.
5. Create an API key in Console, but keep it local only.
6. Use `ARK_API_KEY` as the environment variable name.
7. Run `python3 scripts/doctor.py`.
8. Run a dry run.
9. Review the cost estimate and request preview.
10. Submit only one task with explicit confirmation.

Do not paste API keys into this repository, chat, screenshots, or issue comments.
